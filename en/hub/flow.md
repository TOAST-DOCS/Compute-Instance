## Flow

Flow is a feature that sends messages through various message channels sequentially based on reception results. For example, by sending announcements sequentially through push, email, and SMS based on reception results, you can save costs while increasing reception rates.

1. Click **+ Create Flow**.
2. Enter a flow name and select a delivery purpose. The available message channels and templates are determined based on the configured delivery purpose.
    * If the delivery purpose is advertising, KakaoTalk AlimTalk cannot be used.
    * Only templates that match the delivery purpose can be selected.
3. Click the **+** button to add flow steps.
4. Drag the added flow steps to adjust the delivery order and select message channels.
5. Click **Select** in the template name column and choose the template to use in the flow.
